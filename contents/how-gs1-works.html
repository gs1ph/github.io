GS1 Standards

GS1 Standards create a common foundation for business by uniquely identifying, accurately capturing and automatically sharing vital information about products, locations, assets and more.

Identifying involves using GS1 standards that define unique identification codes (GS1 Identification Keys) which may be used by information systems to refer to real-world entities such as trade items, logistic units, physical locations, documents, service relationships, etc.

Capturing involves using GS1 data capture standards that include definitions of bar code and radio-frequency identification data carriers (RFID) to allow GS1 Identification Keys and supplementary data to be affixed directly to a physical object and standards that specify consistent interfaces to readers, printers, and other hardware and software components that connect the data carriers to business applications.

For sharing: GS1 data standards for master data, business transaction data, and physical event data, as well as communication standards are used to allow trading partners the capability to share the same kind of data to one another. There are also other information-sharing standards such as discovery standards that help locate where relevant data resides across a supply chain and trust standards that help establish the conditions for sharing data with adequate security.

With this data, Businesses will be able to use different GS1 standards in order to streamline business processes such as traceability.
